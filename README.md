# goenv

[GitHub](https://github.com/FlyingDuck/goenv)

---


个人在Go语言学习过程中，当有多个工作空间时，每次都需要修改环境变量，甚是麻烦，借鉴Python virtualenv的思想，使用goenv 用于激活本地开发环境。

用法：

1. 将`active.env`拷贝至工作空间目录下；
2. 激活环境：  
  ```
  $ source active.env 
  ```
3. 退出环境：  
  ```
  $ deactive
  ```
